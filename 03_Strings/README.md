# Strings in C – DSA Practice

This folder contains fundamental **string manipulation problems implemented in C** as part of my **DSA learning journey**.

The goal of this section is to understand how strings work at a low level in C and build a strong foundation before moving to advanced data structures.

---

## Topics Covered

The following core string concepts are implemented:

1. String Length
2. Reverse String
3. Palindrome Check
4. Character Frequency Count
5. Anagram Check

---

## Concepts Practiced

* Character arrays in C
* String traversal using loops
* ASCII character manipulation
* Frequency counting
* Logical problem solving
