# 07_Linked_List

## Overview

This folder contains implementations of **Linked List data structure in C**.

A Linked List is a **linear data structure** where elements (nodes) are connected using **pointers**.
Unlike arrays, linked lists do not store elements in contiguous memory locations.

Each node in a linked list contains:

* **Data** → The value stored in the node
* **Next Pointer** → Address of the next node

Linked Lists are useful when dynamic memory allocation and frequent insertions or deletions are required.

---

## Operations Implemented

### 1. Create Linked List

Creates a linked list by dynamically allocating nodes and linking them together.

### 2. Traverse Linked List

Displays all elements of the linked list by moving from the head node to the last node.

### 3. Insert Node

Insert a new node into the linked list at a specific position.

### 4. Delete Node

Remove a node from the linked list.

### 5. Reverse Linked List

Reverse the order of nodes in the linked list.
