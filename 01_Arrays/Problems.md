# Arrays in C — Data Structures and Algorithms

This folder contains implementations of fundamental **Array problems in C** as part of my **DSA learning journey**.

I am currently building strong **Data Structures & Algorithms foundations** before diving deep into **problem solving in C++ / Python later**.

The goal of this section is to understand:

* Array traversal
