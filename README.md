# 📚 DSA-IN-C

A structured collection of **Data Structures and Algorithms implemented in C**, built while learning and practicing consistently.

This repository documents my journey of understanding DSA from scratch using official documentation, textbooks, and carefully selected YouTube resources, along with hands-on coding practice.

---

## 🎯 Why This Repository?

As a Computer Science student, I wanted to build strong fundamentals instead of just jumping between technologies.

C helps me understand:

* Memory management
* Pointers and low-level logic
* How data structures actually work internally

This repo is my way of:

* Practicing consistently
* Writing clean and understandable implementations
* Tracking my growth over time

---

## 🗂️ Topics Covered

* Arrays
* Strings
* Searching Algorithms
* Sorting Algorithms
* Recursion
* Linked Lists
* Stack
* Queue
* Trees (Basics)

Each topic is organized into separate folders for clarity and easy revision.

---

## 🛠️ My Learning Approach

For every data structure or algorithm:

* I first understand the theory through documentation and trusted learning resources.
* Then I implement it from scratch in C without blindly copying solutions.
* I focus on writing readable, structured code with proper functions.
* I try to analyze time complexity and handle edge cases.

This is not just a collection of programs — it reflects active learning and real understanding.
