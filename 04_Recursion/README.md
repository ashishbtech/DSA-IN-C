# 04 – Recursion

## Overview

This folder contains beginner-level recursion problems implemented in **C**.
Recursion is a programming technique where a function **calls itself** to solve smaller parts of a problem.

Instead of solving the problem using loops, recursion breaks the problem into **smaller subproblems** until it reaches a **base case**.

Learning recursion helps build the foundation for many advanced topics in Data Structures and Algorithms such as:

* Backtracking
* Divide and Conquer
* Tree and Graph algorithms
* Dynamic Programming

---

## What I Am Learning Here

In this section I am focusing on:

* Understanding **how recursion works**
* Identifying the **base case**
* Understanding the **recursive call**
* Visualizing the **function call stack**
* Converting simple loop problems into recursion

---

## Problems Covered

1. `print_numbers_1_to_n.c`
   Print numbers from **1 to N** using recursion.

2. `print_numbers_n_to_1.c`
   Print numbers from **N to 1** using recursion.

3. `factorial_recursion.c`
   Calculate **factorial of a number** using recursion.

4. `fibonacci_recursion.c`
   Generate **Fibonacci sequence** using recursion.

5. `sum_of_n_numbers.c`
   Find the **sum of first N natural numbers** using recursion.

## Time and Space Considerations

Most basic recursion problems have:

* **Time Complexity:** O(n)
* **Space Complexity:** O(n) due to the function call stack.
